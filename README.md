# Repositorio de Diseño Electrónico y Sistemas Embebidos

Este repositorio contiene los proyectos y exámenes prácticos desarrollados, enfocados en el diseño de hardware profesional, captura esquemática, ruteo de PCBs y programación de microcontroladores ARM Cortex-M.


<br>


## **Menú Principal de Proyectos**


<br>


El contenido está organizado cronológicamente por hitos académicos, integrando herramientas de diseño asistido por computadora (EDA) y entornos de desarrollo embebido:


<br>


### **Parcial 1: Diseño de Interfaz de Visualización**

* **Parcial1_Display**: Implementación de un sistema de visualización para un **reloj con alarma**.
* **Documentación**: El archivo `P1_KS.pdf` contiene los diagramas, la lógica de funcionamiento y el reporte técnico del sistema.


<br>


### **Parcial 2: Desarrollo de Sistemas y Software Embebido**

* **DSN - KitSTM32**: Archivos de diseño esquemático (`KIT_STM32.DSN`) desarrollados en OrCAD Capture para el sistema basado en STM32.
* **JuegoSTM**: Desarrollo del juego **"Rock, Paper, Scissors, Lizard, Spock"** programado para el microcontrolador **STM32L053R8**.
    * Incluye la estructura profesional de carpetas: `Inc` (headers), `Src` (source code) y `Startup`.
    * Configuración completa para el entorno de desarrollo basada en archivos `.project` y `.cproject`.


<br>


### **Parcial Final: Hardware Development Kit (HDK)**

* **Kit-STM32**: Diseño integral de una placa de desarrollo basada en el microcontrolador **STM32G071RBT6**.
* **Recursos**: Incluye archivos de fabricación (Gerbers), diseño de PCB en Allegro (`.brd`), esquemáticos y análisis de costos.
* **Detalles**: Para más información, consulte el [README específico de la carpeta ParcialFinal](./ParcialFinal/README.md).


<br>


## **Herramientas y Tecnologías**


<br>


* **Diseño de Hardware**: Cadence OrCAD Capture (Esquemáticos) y Allegro PCB Editor (Layout).
* **Desarrollo de Software**: STM32CubeIDE / Entornos basados en Eclipse para microcontroladores STM32.
* **Lenguajes**: C para sistemas embebidos.
* **Documentación**: Reportes técnicos detallados y gestión de Bill of Materials (BOM).
